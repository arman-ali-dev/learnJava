### 1. Create an ArrayList of integers and add five numbers to it.

```java

    public static void main(String[] args) {

        nums.add(1);
        nums.add(2);
        nums.add(3);
        nums.add(4);
        nums.add(5);

        System.out.println(nums);
    }
}
```
